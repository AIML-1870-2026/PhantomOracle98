How to read this data

Source: All data comes from the FDA's OpenFDA Adverse Event Reporting System (FAERS). Reports are submitted voluntarily by patients, healthcare providers, and manufacturers.

What are adverse events? An adverse event is any undesirable experience associated with using a medical product. It does not necessarily mean the drug caused the problem — only that it was reported while taking the drug.

Serious outcomes include hospitalization, disability, life-threatening events, or death. These are distinguished from non-serious outcomes in the severity charts.

Report counts do not represent how common a reaction is in the general population. A drug taken by millions may have more reports simply because more people use it.

Drug classes are approximate groupings based on common medication types. Comparisons within a class can reveal relative safety profiles, but always consult a healthcare professional.

Data currency: OpenFDA is updated quarterly. Some recent events may not yet appear.
